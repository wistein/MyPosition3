 <h4>History</h4
 <B>My Position, version 1.3.1 by wistein, 2017-08-19,</B><br>
 https://github.com/wistein/MyPosition3<br>
 Height data is corrected according to Geoid sealevel.<br>
 <br>
 <B>My Position, version 1.3.0 by wistein, 2017-04-11,</B><br>
 includes code corrections and enhancements,<br>
 interrupts GPS usage when app pauses or ends GPS usage when app terminates,<br>
 uses Reverse Geocoding by OpenStreetMap (OSM) query for showing address info,<br>
 therefore uses optionally your provided email address as recommended by OSM,<br>
 country-specific representation of address info (at, ch, de, fr, it, rest of the world),<br>
 introduces localized strings (German, English) and<br>
 considers localized number formats in calculations.<br>
